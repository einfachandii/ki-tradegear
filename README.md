# 🤖 Musterdepot KI Agent

Autonomes Trading-System mit 3.000€ Startkapital.

## 📊 Live-Dashboard

**URL:** [https://andreashergett.github.io/musterdepot-ki-agent/](https://andreashergett.github.io/musterdepot-ki-agent/)

## Features

- ✅ **Autonomes Trading**: KI führt selbstständig Analysen durch und handelt
- ✅ **Echtzeit-Daten**: Aktuelle Kurse und Portfolio-Performance
- ✅ **Transparenz**: Alle Trades mit KI-Begründungen dokumentiert
- ✅ **Mobile-optimiert**: Responsive Design für alle Geräte
- ✅ **Auto-Update**: Daten werden während Börsenzeiten (8-22 Uhr) alle 5 Minuten aktualisiert

## Strategie

**Asset-Klassen:**
- US & EU Aktien (Tech, Defense, Healthcare, Value)
- Indizes (S&P 500, DAX, etc.)
- Rohstoffe (Gold, Öl, etc.)
- Optional: Derivate mit 2-3x Hebel

**Handelszeiten:**
- Montag-Freitag, 8-22 Uhr (wie Scalable Capital)

**Risiko-Management:**
- Stop-Loss bei -8% pro Position
- Max. 60% investiert, 40% Cash-Reserve
- Diversifikation über Sektoren

## Tech Stack

- Frontend: Vanilla HTML/CSS/JavaScript
- Hosting: GitHub Pages
- Updates: Automatisch via Git Push (dorabot Mac)
- Backend: Python Trading-Bot (lokal)
